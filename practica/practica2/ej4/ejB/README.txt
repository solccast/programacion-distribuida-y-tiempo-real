# Uso 
> se requiere Vagrant
- En el directorio actual ejecutamos 
`vagrant up `.
- Una vez finalizada, usando vagran ssh vm1` se nos abrirá la conexión vía ssh a la máquina virtual. En la shell ejecutamos ./run_server.sh
- Para levantar el cliente es necesario correr el script `/run_client.sh` el cual por defecto viene seteada con la ip de la máquina virtual. 
- Los resultados se verán reflejados en el archivo `ej4b_resultados.txt` en el mismo directorio donde se encuentre.
